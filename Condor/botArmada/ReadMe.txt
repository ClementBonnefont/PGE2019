repository botArmada

Nous avons ici 2 programmes utilisant le logiciel armada du dossier Build.
setCoord.py permet de set les coordonnées de tous les boutons du logiciel (utilisé lorsque l'on change l'écran).
botArmada.py permet de lancer le bot pour créer des BDD. Les paramètres utilisés sont données lignes 340 et 351 (doit être modifiés à cet endroit).

Le logiciel armada doit être mis dans ce dossier à l'image du dossier build donné en exemple.

ATTENTION : le bot prend le contrôle de votre poste et celui-ci devient inutilisable pendant l'execution.
	Le bot prend le controle de votre souris et clavier, un non-respect des instructions peut faire planter le bot et modifier les actions de vos périphériques (un redémarrage est alors nécessaire).
