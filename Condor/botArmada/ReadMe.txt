repository botArmada

Nous avons ici 2 programmes utilisants le logiciel armada du dossier Build.
setCoord.py permet de set les coordonées de tous les boutons du logiciel (utilisé lorsque l'on change l'écran).
botArmada.py permet de lancer le bot pour créé des BDD. Les paramètres utilisés sont données lignes 340 et 351 (doit être modifié à cet endroit).

ATTENTION : le bot prend le controle de votre poste et celui-ci devient inutilisable pendant l'execution.
	Le bot prend le controle de votre souris et clavier, un non-respet des instructions peu faire planter le bot et modifier les actions de vos périphériques (un redémarrage est alors nécessaire).
