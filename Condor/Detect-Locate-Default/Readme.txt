repository Detect-Locate-Default

Ici se trouvent les travaux concernant la détection des défauts : 
	- Dans le dossier 'darkflow' se trouve le réseau de neurones Yolo et ces entrainements.
	- Dans le dossier 'integration' se trouve tous les algorithmes de détection de rouille, nid d'oiseau et isolateur cassé

Aussi, les models entrainés du Faster RCNN sur la detection des Isolateurs ainsi que le tutoriel utilisé.
Tous les repertoires qui ont servi à la réalisation du Tuto sont aussi joins.

NB: Nous n'avons pas intégré le faster RCNN dans la solution final, car nous avons prilegier YOLO V2 pour ça portabilité.
