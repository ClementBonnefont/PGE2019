repository condor

Ici se trouve les travaux de l'équipe Condor. 
Nous avons décidé de le séparer en deux parties:
	- Dans le dossier 'Detect-Locate_Pylon' se trouvent tous les algorithmes et méthodes développés permettant l'identification du type de pylône, ainsi que la localisation du drone par rapport au pylône.
	- Dans le dossier 'Detect-Locate_Default' se trouvent tous les algorithmes et méthodes concernant la détection des défauts : rouille, nid d'oiseau et isolateur cassé
	- Dans le dossier 'botArmada' se trouvent tous les algorithmes permettant de créer automatiquement des bases de données à l'aide du logiciel armada
